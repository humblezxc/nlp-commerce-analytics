# E-commerce Report Generator (PoC)

A Proof of Concept for an e-commerce report generator that allows store owners to create and manage reports using natural language queries or predefined templates.

## Features

- JWT-based authentication
- Multi-tenancy with Row-Level Security (RLS)
- Natural language report generation using AI
- Interactive data visualizations
- Responsive UI for desktop and tablet

## Tech Stack

### Backend
- Node.js + Express
- SQLite + Sequelize ORM
- JWT Authentication
- OpenAI API for NLP

### Frontend
- React + Vite
- Tailwind CSS + shadcn/ui
- Recharts for visualizations

## Project Structure

```
├── backend/          # Express API server
├── frontend/         # React SPA
└── README.md
```
